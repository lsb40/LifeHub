# LifeHub

LifeHub is a full-stack web application that helps users track their health and lifestyle habits in one place. Users can log meals, track nutrition, steps, water intake, and view insights over time. The goal is to provide a simple, personalized platform for managing overall well-being.

## Features

- Search and log foods using Spoonacular API
- Track calories, protein, carbs, and fats
- Log daily steps and water intake
- View a rotating 30-day meal plan
- Visual dashboard with daily and weekly summaries
- FastAPI backend with RESTful endpoints
- React-based frontend for a responsive experience

## Tech Stack

- Frontend: React, JavaScript, HTML, CSS, Bootstrap
- Backend: Python, FastAPI
- API Integration: Spoonacular
- Optional: SQLite or PostgreSQL for persistent data
